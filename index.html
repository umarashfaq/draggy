Draggy - A jQuery Scrollbar Plugin

Overview

Draggy is a lightweight jQuery plugin that allows to conveniently create GMail style scrollbars.

Quick Start

Using Draggy in your project is simple: just include jquery.draggy.js and jquery.draggy.css in your HTML page and invoke following code:

$('#my-element').draggy();

Theoretical Detail

You can invoke draggy on any DOM element you like. Draggy will automatically calculate if the element is overflowing out of it's parent's dimensions and attach scrollbars accordingly.

Target Width Greater Than Parent Width

When target width is greater than it's parent width, draggy will attach a horizontal scrollbar with the parent.

Target Height Greater Than Parent Height

When target height is greater than it's parent height, draggy will attach a vertical scrollbar with the parent.

Target Width And Height Greater Than Parent Width And Height

When target width and height are greater than it's parent's width and height, draggy will attach horizontal and vertical scrollbars with the parent.

API

Construction

As shown before, applying draggy is simple:

  1. Just select an element with jQuery and make sure that its dimensions (width or height) are greater than those of it's parent.
  2. Invoke draggy() on it.

$(selector).draggy();

IMPORTANT: Draggy should be instantiated on an element after all of it's content has been loaded. So if the element contains any images, draggy should be instantiated on it after images have been loaded. jQuery's $(document).ready(callback) invokes a callback when only DOM has been loaded; it doesn't wait for images. Thankfully, we have another function, $(window).load(callback) which invokes the callback after all of it's images have been downloaded.

Destruction

$(selector).draggy('kill');

Configuration

Draggy is highly configurable. You can easily tweak it's behavior using one of the following properties. To apply a property, simply pass it on with a value in constructor.

$(selector).draggy({
	'draggability' : 'both'
});
  • draggability 'horizontal' | 'vertical' | 'both' | 'none' (default)
  • mousewheel 'horizontal' | 'vertical' | 'both' | 'none' (default)

    It requires jquery.mousewheel.js.

  • arrowkeys 'horizontal' | 'vertical' | 'both' | 'none' (default)

    For now, you can associate arrowkeys with only one instance of draggy on a page. If you try to associate arrowkeys with multiple instances, they will only work for the instance they got associated with in the last.

  • scrolls 'horizontal' | 'vertical' | 'both' | 'none' | 'auto' (default)
